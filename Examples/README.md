# Proof Sketcher Examples

This directory contains example proofs demonstrating Proof Sketcher's capabilities.

## Files

- [Basic.lean](Basic.lean) - Introductory examples showing core tactics

## Running Examples

```bash
lake build Examples
```

Or open the files in VS Code with the Lean 4 extension to see interactive results in the Infoview.

## Learning Path

1. Start with [Basic.lean](Basic.lean) to understand `sketch`, `milestone`, and `outline`
2. Experiment by modifying existing examples
3. Create your own proofs using the patterns shown
